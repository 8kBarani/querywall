## Project Summary
Querywall is a Next.js application designed for efficient querying and data management.

## Tech Stack
- Framework: Next.js (App Router)
- Language: TypeScript
- Styling: Tailwind CSS
- Runtime: Bun/Node.js
- Key Dependencies: Framer Motion, Radix UI, Lucide Icons, Stripe, Drizzle ORM

## Architecture
- `src/app/`: Next.js App Router pages and layouts
- `src/components/`: Reusable UI components
- `src/lib/`: Utility functions and shared logic
- `src/hooks/`: Custom React hooks
